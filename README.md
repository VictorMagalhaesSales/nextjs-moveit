## How to run the project?
```bash
# Clone this repository
$ git clone https://github.com/VictorMagalhaesSales/nextjs-moveit

# Access the repository on your terminal
$ cd nextjs-moveit

# Install dependencies
$ yarn install

# Run
$ yarn dev

# The API will be running on port 3000
```
